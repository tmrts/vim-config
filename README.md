Installation
------------

To install, run

    git clone git:https://github.com/TamerTas/vim-config.git ~/.vim
    sh ~/.vim/install.sh

Bundles
------------

.vimrc.bundles file is used to keep the list of plugins in a clear manner.
Use it for plugin purposes
